Step 3: Build and restart services

# Stop current services
docker compose down

# Start services (this will rebuild images if needed)
docker compose up -d

# Wait for containers to be healthy, then run migrations
docker compose exec laravel php artisan migrate --seed
Step 4: Build the React app

# Option A: Build locally and copy dist folder
cd apps/web
npm install
npm run build
# Copy dist folder to VPS if building locally

# Option B: Build inside the container (if Docker is configured for this)
docker compose exec frontend npm run build


If there's any database stuff:
docker exec -it laravel-php php artisan migrate
